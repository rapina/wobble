# SKILLS.md

서바이버 모드의 스킬 목록과 시각적 특징을 정리한 문서입니다.

## 스킬 시스템 개요

### 활성화 타입 (Activation Type)

| 타입 | 설명 | 시각적 특징 |
|------|------|-------------|
| `passive` | 항상 활성화, 발사체/스탯 수정 | 쿨다운 없음, 별도 시각 효과 없음 |
| `aura` | 플레이어 주변 지속 효과 | 원형 오라/글로우 표시 |
| `active` | 주기적 발동 | 쿨다운 표시, 발동 시 시각 효과 |

### 스킬 카테고리

| 카테고리 | 설명 |
|----------|------|
| `projectile` | 발사체 수정 (튕김, 관통, 폭발, 분산) |
| `aura` | 지속 오라 효과 (복사, 혼돈장, 반발) |
| `orbital` | 공전 오브젝트 (궤도 폭격, 회전참) |
| `player` | 플레이어 상태 수정 (고스트, 리듬) |
| `trigger` | 이벤트 트리거 (처치 시 연쇄, 열 전도) |

### 스킬 의존성 시스템 (Tags & Requires)

스킬은 **베이스 스킬**과 **모디파이어 스킬**로 구분됩니다.

| 구분 | 설명 | 예시 |
|------|------|------|
| **베이스 스킬** | 독립적으로 동작, `tags` 제공 | 운동 탄환 → `projectile` 태그 제공 |
| **모디파이어 스킬** | `requires` 조건 필요, 베이스 강화 | 탄성 충돌 → `projectile` 태그 필요 |

**태그 (Tags):**
- `projectile`: 발사체 생성 기능
- `wave`: 파동 생성 기능
- `orbital`: 공전 오브젝트 기능
- `chain`: 연쇄 공격 기능

## Projectile 스킬 (발사체)

### 🌟 운동 탄환 (Kinetic Shot) - 베이스 스킬
- **ID**: `kinetic-shot`
- **물리 공식**: 운동에너지 (KE = ½mv²)
- **활성화**: `active` (쿨다운: 0.5초)
- **태그 제공**: `projectile`
- **효과**: 운동에너지를 담은 탄환 발사
- **시각**: 에너지 구체 형태의 발사체
- **레벨업**: 데미지 10→22, 발사속도 2→4회/초
- **스타터**: ✅ 게임 시작 시 선택 가능

---

### 탄성 충돌 (Elastic Bounce) - 모디파이어
- **ID**: `elastic-bounce`
- **물리 공식**: 탄성 충돌 (p₁ + p₂ = p₁' + p₂')
- **활성화**: `passive`
- **필요 조건**: `requires: ['projectile']`
- **효과**: 발사체가 벽/적에 튕김
- **시각**: 발사체가 충돌 시 반사각으로 튕겨나감
- **레벨업**: 튕김 횟수 1→5회 증가

### 운동량 관통 (Momentum Pierce) - 모디파이어
- **ID**: `momentum-pierce`
- **물리 공식**: 운동량 (p = mv)
- **활성화**: `passive`
- **필요 조건**: `requires: ['projectile']`
- **효과**: 발사체가 적을 관통
- **시각**: 발사체가 적을 통과하며 지나감
- **레벨업**: 관통 2→10회, 데미지 감쇠 감소

### 압력파 (Pressure Wave) - 모디파이어
- **ID**: `pressure-wave`
- **물리 공식**: 이상 기체 (PV = nRT)
- **활성화**: `passive`
- **필요 조건**: `requires: ['projectile']`
- **효과**: 발사체 피격 시 폭발
- **시각**: 원형 폭발 파동, 범위 내 적 피해
- **레벨업**: 폭발 반경 50→180 증가

### 진동수 증폭 (Frequency Burst) - 모디파이어
- **ID**: `frequency-burst`
- **물리 공식**: 광전효과 (E = hf)
- **활성화**: `passive`
- **필요 조건**: `requires: ['projectile']`
- **효과**: 발사 속도 증가
- **시각**: 발사체가 더 빠르게 연속 발사
- **레벨업**: 발사 속도 +30%→+150%

### F=ma 충격 (FMA Impact) - 모디파이어
- **ID**: `fma-impact`
- **물리 공식**: 뉴턴 제2법칙 (F = ma)
- **활성화**: `passive`
- **필요 조건**: `requires: ['projectile']`
- **효과**: 데미지/넉백 증가
- **시각**: 발사체가 더 크고 무거워 보임
- **레벨업**: 데미지 +30%→+150%, 넉백 +50%→+150%

### 중력 유도 (Gravity Pull) - 모디파이어
- **ID**: `gravity-pull`
- **물리 공식**: 만유인력 (F = GMm/r²)
- **활성화**: `passive`
- **필요 조건**: `requires: ['projectile']`
- **효과**: 발사체가 적을 향해 휘어짐
- **시각**: 발사체가 곡선 궤적으로 적 추적
- **레벨업**: 추적력 0.5→3.0 증가

### 굴절 분산 (Refraction Spread) - 모디파이어
- **ID**: `refraction-spread`
- **물리 공식**: 스넬 법칙 (n₁sinθ₁ = n₂sinθ₂)
- **활성화**: `passive`
- **필요 조건**: `requires: ['projectile']`
- **효과**: 여러 방향으로 분산 발사
- **시각**: 부채꼴 형태로 여러 발사체
- **레벨업**: 탄환 3→12발, 각도 20°→60°

### 탄성 회귀 (Elastic Return) - 모디파이어
- **ID**: `elastic-return`
- **물리 공식**: 훅의 법칙 (F = -kx)
- **활성화**: `passive`
- **필요 조건**: `requires: ['projectile']`
- **효과**: 발사체가 최대 거리에서 되돌아옴
- **시각**: 스프링처럼 늘어났다 돌아오는 궤적
- **레벨업**: 거리 150→300, 복귀 데미지 50%→100%

---

### 🌟 원심력 펄스 (Centripetal Pulse) - 베이스 스킬
- **ID**: `centripetal-pulse`
- **물리 공식**: 구심력 (F = mv²/r)
- **활성화**: `active` (쿨다운: 5초)
- **효과**: 주기적 충격파로 적 밀어냄
- **시각**: 플레이어 중심 원형 펄스 파동
- **레벨업**: 주기 5→2초, 반경 80→200, 넉백 100→300
- **스타터**: ✅ 게임 시작 시 선택 가능

## Aura 스킬 (오라)

### 복사 오라 (Radiant Aura)
- **ID**: `radiant-aura`
- **물리 공식**: 슈테판-볼츠만 법칙 (P = σAT⁴)
- **활성화**: `aura`
- **효과**: 주변 적에게 지속 데미지
- **시각**: 플레이어 주변 따뜻한 색상의 원형 글로우
- **레벨업**: 반경 50→100, 데미지 3→18/초

### 혼돈장 (Chaos Field)
- **ID**: `chaos-field`
- **물리 공식**: 엔트로피 (ΔS > 0)
- **활성화**: `aura`
- **효과**: 적 이동 경로 무작위 교란
- **시각**: 보라색 불규칙한 입자들이 떠다니는 필드
- **레벨업**: 반경 60→150, 교란 20%→80%

### 자기장 방어 (Magnetic Shield)
- **ID**: `magnetic-shield`
- **물리 공식**: 로렌츠 힘 (F = qvB)
- **활성화**: `aura`
- **효과**: 적 경로를 휘게 함
- **시각**: 파란색 자기장 링, 적이 휘어짐
- **레벨업**: 반경 60→150, 편향 30%→100%

### 정전기 반발 (Static Repulsion)
- **ID**: `static-repulsion`
- **물리 공식**: 쿨롱 법칙 (F = kq₁q₂/r²)
- **활성화**: `aura`
- **효과**: 적을 지속적으로 밀어냄
- **시각**: 전기 스파크, 적이 밀려남
- **레벨업**: 반경 80→180, 밀어냄 50→200

### 시간 왜곡 (Time Warp)
- **ID**: `time-warp`
- **물리 공식**: 시간 지연 (t = t₀/√(1-v²/c²))
- **활성화**: `aura`
- **효과**: 주변 적 속도 감소
- **시각**: 어두운 시공간 왜곡 필드
- **레벨업**: 반경 80→180, 감속 20%→60%

### 유체 흐름 (Flow Stream)
- **ID**: `flow-stream`
- **물리 공식**: 베르누이 원리 (P + ½ρv² + ρgh = const)
- **활성화**: `aura`
- **효과**: 빠른 흐름으로 적 끌어당김
- **시각**: 청록색 유체 흐름 시각화
- **레벨업**: 속도 100→250, 흡입력 30→150

### 자기 흡인 (Magnetic Pull)
- **ID**: `magnetic-pull`
- **물리 공식**: 자기장 (B = μ₀I/2πr)
- **활성화**: `aura`
- **효과**: 적/아이템을 끌어당김
- **시각**: 자석 인력선 시각화
- **레벨업**: 반경 80→170, 흡인력 30→140

## Orbital 스킬 (공전)

### 궤도 폭격 (Orbital Strike)
- **ID**: `orbital-strike`
- **물리 공식**: 케플러 제3법칙 (T² ∝ a³)
- **활성화**: `aura`
- **효과**: 공전 위성이 적에게 데미지
- **시각**: 플레이어 주변을 도는 작은 구체들
- **레벨업**: 위성 1→5개, 반경 60→100, 데미지 10→22

### 토크 회전참 (Torque Slash)
- **ID**: `torque-slash`
- **물리 공식**: 토크 (τ = rF sin θ)
- **활성화**: `aura`
- **효과**: 회전하는 칼날이 적 타격
- **시각**: 플레이어 주변을 빠르게 회전하는 칼날/호
- **레벨업**: 반경 60→130, 데미지 8→30, 회전 1.5→3회/초

### 🌟 플라즈마 방전 (Plasma Discharge) - 베이스 스킬
- **ID**: `plasma-discharge`
- **물리 공식**: 전기 방전
- **활성화**: `active` (쿨다운: 3초)
- **효과**: 적들을 연결하는 체인 라이트닝
- **시각**: 시안색 번개가 적들 사이를 연결
- **레벨업**: 데미지 15→55/초, 연쇄 2→7명, 사거리 200→350
- **스타터**: ✅ 게임 시작 시 선택 가능

## Player 스킬 (플레이어 상태)

### 양자 터널링 (Quantum Tunnel)
- **ID**: `quantum-tunnel`
- **물리 공식**: 양자 터널링
- **활성화**: `active` (쿨다운: 8초)
- **효과**: 고스트 모드로 적 통과 + 데미지
- **시각**: 플레이어가 반투명, 잔상 효과
- **레벨업**: 쿨다운 8→4초, 지속 0.5→1.5초, 데미지 10→40

### 진자 리듬 (Pendulum Rhythm)
- **ID**: `pendulum-rhythm`
- **물리 공식**: 진자 주기 (T = 2π√(L/g))
- **활성화**: `passive`
- **효과**: 주기적 공격력 증가
- **시각**: 공격력 UI가 사인파처럼 오르내림
- **레벨업**: 주기 4→2초, 최대 보너스 +50%→+200%

### 부력 폭탄 (Buoyant Bomb)
- **ID**: `buoyant-bomb`
- **물리 공식**: 부력 (F = ρVg)
- **활성화**: `passive`
- **효과**: 발사체가 떠올랐다 폭발하며 낙하
- **시각**: 발사체가 위로 뜬 후 떨어지며 폭발
- **레벨업**: 체공 1.5→0.7초, 범위 60→160, 데미지 20→100

### 🌟 파동 펄스 (Wave Pulse) - 베이스 스킬
- **ID**: `wave-pulse`
- **물리 공식**: 파동 방정식 (v = fλ)
- **활성화**: `active` (쿨다운: 2초)
- **효과**: 사인파 형태로 퍼지는 데미지
- **시각**: 플레이어에서 퍼지는 물결 링
- **레벨업**: 파장 50→100, 진폭 20→60, 속도 100→200
- **스타터**: ✅ 게임 시작 시 선택 가능

### 맥놀이 펄스 (Beat Pulse)
- **ID**: `beat-pulse`
- **물리 공식**: 맥놀이 (f_beat = |f₁ - f₂|)
- **활성화**: `passive`
- **효과**: 두 주파수 간섭으로 주기적 강화
- **시각**: 맥놀이 패턴의 시각적 강조
- **레벨업**: 맥놀이 1→3Hz, 증폭 +20%→+70%

## Trigger 스킬 (트리거)

### 붕괴 연쇄 (Decay Chain)
- **ID**: `decay-chain`
- **물리 공식**: 방사성 붕괴 (N = N₀e^(-λt))
- **활성화**: `passive`
- **효과**: 적 처치 시 확률적 연쇄 폭발
- **시각**: 녹색 방사성 폭발, 연쇄 반응
- **레벨업**: 확률 15%→40%, 반경 60→130

### 열 전도 체인 (Heat Chain)
- **ID**: `heat-chain`
- **물리 공식**: 열전도 (Q/t = kA(ΔT/Δx))
- **활성화**: `passive`
- **효과**: 데미지가 인접 적에게 전도
- **시각**: 주황색 열 전달 라인
- **레벨업**: 범위 50→100, 전달 30%→70%, 연쇄 2→7회

### 주파수 편이 (Frequency Shift)
- **ID**: `frequency-shift`
- **물리 공식**: 도플러 효과 (f' = f(v ± vr)/(v ∓ vs))
- **활성화**: `passive`
- **효과**: 접근하는 적에게 추가 데미지
- **시각**: 접근 적은 붉게, 멀어지는 적은 푸르게
- **레벨업**: 접근 +20%→+70%, 후퇴 -10%→-30%

### 탈출 속도 폭발 (Escape Burst)
- **ID**: `escape-burst`
- **물리 공식**: 탈출 속도 (v = √(2GM/r))
- **활성화**: `passive`
- **효과**: 충분한 속도로 이동 시 폭발 발생
- **시각**: 녹색 속도 궤적, 폭발 링
- **레벨업**: 임계속도 300→100, 보너스 +50%→+200%, 반경 50→120

## 스타터 스킬 (레벨 1 선택 가능)

게임 시작 시 선택 가능한 **베이스 스킬**:

| 스킬 | ID | 빌드 타입 | 연계 가능 모디파이어 |
|------|-----|-----------|---------------------|
| 🌟 **운동 탄환** | `kinetic-shot` | 발사체 빌드 | 튕김, 관통, 폭발, 유도 등 |
